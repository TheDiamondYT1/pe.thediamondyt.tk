## Resource Pack Info

---

|Host|Authentication|
|----|--------------|
|xforge.xboxlive.com:443|No|
  
### Constructing the request
```
GET /v1/catalog/items/{id}/
```

##### Explained
`{id}` is the id of the resource pack

For example, for the Plastic Texture Pack: `G009SWJHK75F`
  
##### Headers
```http
Content-Type: application/json
```
