## List Worlds

|Method|URL|
|------|---|
|GET|/worlds|
  
### Example Request

```
GET /worlds HTTP/1.1
Host: pocket.realms.minecraft.net:443
Authorization: (long xbox live token)
Cache-Control:no-cache
Charset:utf-8
Client-Version:1.0.4
User-Agent:MCPE/Android
Connection: Keep-Alive
```
